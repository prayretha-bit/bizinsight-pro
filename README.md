# BizInsight Pro

Interactive Sales Analytics Dashboard built using Streamlit.

### 🔗 Live App
[Open Dashboard](https://bizinsight-pro-5bfebtyygudnckdp7aa9ny.streamlit.app)

### 📊 Features
- Sales overview (Revenue, Profit, Quantity)
- Region-wise Analysis
- Category-wise Insights
- Interactive Filters

### 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Plotly / Matplotlib
